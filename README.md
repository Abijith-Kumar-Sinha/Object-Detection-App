# 📱 Object Detection App

An Android application that performs real-time object detection using TensorFlow Lite.

## Features
- Real-time object detection using device camera
- On-device inference (no internet required)
- Bounding box visualization with labels
- Optimized for mobile performance

## Tech Stack
- Kotlin
- TensorFlow Lite
- CameraX
- Android SDK

## How It Works
The app captures live camera frames using CameraX, processes them through a TensorFlow Lite model, and displays detected objects with bounding boxes on the screen.

## Use Cases
- Smart surveillance
- Educational AI demos
- Mobile computer vision projects

## Performance
Runs fully on-device for low latency and efficient processing.
